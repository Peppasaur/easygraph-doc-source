easygraph.DiGraph.in_degree¶

DiGraph.in_degree(weight='weight')[source]¶

Returns the weighted in degree of each node.

Parameters

weight (string, optinal (default : 'weight')) – Weight key of the original weighted graph.

Returns

in_degree – Each node’s (key) weighted in degree (value).

Return type

dict

Notes

If the graph is not weighted, all the weights will be regarded as 1.

See also

out_degree(), degree()

Examples

>>> G.in_degree(weight='weight')
