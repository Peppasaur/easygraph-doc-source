Datasets

EasyGraph provides API to generate graph instance of several open dataset.

The graph dataset includes blogcatalog, Flickr, etc.

KarateClub

easygraph.datasets.get_graph_karateclub()[source]

Returns the undirected graph of Karate Club.

Returns

get_graph_karateclub – The undirected graph instance of karate club from dataset: http://vlado.fmf.uni-lj.si/pub/networks/data/Ucinet/UciData.htm

Return type

easygraph.Graph

References

1

http://vlado.fmf.uni-lj.si/pub/networks/data/Ucinet/UciData.htm
