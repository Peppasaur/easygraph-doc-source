Structural Hole Spanners

EasyGraph has implemented several most commonly used structural holes spanners detection methods, including HIS, MaxD, HAM, Common_Greedy, AP_Greedy. Burt’s evaluation metric for structural holes, Effective Size, Efficiency, Constraint and Hierarchy, are also included.

HIS

get_structural_holes_HIS(G, C[, epsilon, weight])

Structural hole spanners detection via HIS method.

MaxD

get_structural_holes_MaxD(G, k, C)

Structural hole spanners detection via MaxD method.

AP_Greedy

common_greedy(G, k[, c, weight])

Common greedy method for structural hole spanners detection.

AP_Greedy(G, k[, c, weight])

AP greedy method for structural hole spanners detection.

ICC

ICC(G, k)

an efficient algorithm for structural hole spanners detection.

BICC(G, k, K, l)

an efficient algorithm for structural hole spanners detection.

AP_BICC(G, k, K, l)

an efficient algorithm for structural hole spanners detection.

HAM

get_structural_holes_HAM(G, k, c, …)

Structural hole spanners detection via HAM method.

NOBE

NOBE_SH(G, K, topk)

detect SH spanners via NOBE[1].

NOBE_GA_SH(G, K, topk)

detect SH spanners via NOBE-GA[1].

WeakTie

weakTie(G, threshold, k)

Return top-k nodes with highest scores which were computed by WeakTie method.

weakTieLocal(G, edges_plus, edges_delete, …)

Find brokers in evolving social networks, utilize the 2-hop neighborhood of an affected node to identify brokers.
