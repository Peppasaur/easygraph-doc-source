easygraph.DiGraph.add_edges_from_file¶

DiGraph.add_edges_from_file(file, weighted=False)[source]¶

Added edges from file For example, txt files,

Each line is in form like: a b 23.0 which denotes an edge a → b with weight 23.0.

Parameters

  • file (string) – The file path.

  • weighted (boolean, optional (default : False)) – If the file consists of weight infomation, set True. The weight key will be set as ‘weight’.

Examples

If ./club_network.txt is:

Jack Mary 23.0

Mary Tom 15.0

Tom Ben 20.0

Then add them to G

>>> G.add_edges_from_file(file='./club_network.txt', weighted=True)
