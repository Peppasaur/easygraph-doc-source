Graph embedding via SDNE.

Parameters

  • graph (easygraph.Graph, easygraph.DiGraph) –

  • hidden_size (list of two elements, optional (default : [32, 16])) – The hidden size.

  • alpla (float, optional (default : 1e-6)) – The alpha value in [1].

  • beta (float, optional (default : 5.)) – The beta value in [1].

  • nu1 (float, optional (default : 1e-5)) – The nu1 value in [1].

  • nu2 (float, optional (default : 1e-4)) – The nu2 value in [1].

Examples

>>> model = SDNE(G,
    ...          hidden_size=[256, 128]) # The hidden size in SDNE.
>>> model.train(batch_size=3000, epochs=40, verbose=2)
>>> embeddings = model.get_embeddings() # Returns the graph embedding results.
