easygraph.readwrite.gexf.relabel_gexf_graph

easygraph.readwrite.gexf.relabel_gexf_graph(G)[source]

Relabel graph using “label” node keyword for node label.

Parameters

G (graph) – a graph read from GEXF data

Returns

H – a graph with relabeled nodes

Return type

graph

Raises

EasyGraphError – If node labels are missing or not unique while relabel=True.

Notes

This function relabels the nodes in a graph with the “label” attribute. It also handles relabeling the specific GEXF node attributes “parents”, and “pid”.
